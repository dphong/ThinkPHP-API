插入数据接口

请求url

http://api.com/api/map/add     // POST请求


请求参数

参数名 参数含义 类型 备注
lng 经度 float 最多6位小数，3位整数，必选
lat 纬度 float 最多6位小数，3位整数，必选
apikey 用户的访问权限key string(32) 必选


响应参数（json格式）

参数名 参数含义 类型 备注
status 状态码 int 1 : 成功
-1 : 失败
message 响应的信息 int 200 : successful
201 : APIKey error
202 : lng error
203 : lat error
204 : data insertion failure
