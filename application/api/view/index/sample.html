插入数据接口(POST)

请求url

https://whark.cn/api/map/add     // POST请求


请求参数

参数名 参数含义 类型 备注
lng 经度 float 最多6位小数，3位整数，必选
lat 纬度 float 最多6位小数，3位整数，必选
apikey 用户的访问权限key string(32) 必选


响应参数（json格式）

参数名 参数含义 类型 备注
status 状态码 int 1 : 成功,并在message中返回id
-1 : 失败
message 响应的信息 int 201 : APIKey error
202 : lng error
203 : lat error
204 : data insertion failure
205 : APIKey is empty

查询数据接口(POST)

请求url

https://whark.cn/api/map/get     // POST请求


请求参数

参数名 参数含义 类型 备注
apikey 用户的访问权限key string(32) 必选


响应参数（json格式）

参数名 参数含义 类型 备注
status 状态码 int 1 : 成功
-1 : 失败
message 响应的信息 int 200 : successful
201 : APIKey error
202 : Data is empty
203 : query failure
data map_id map表的主键 int
lng 经度 float
lat 纬度 float
create_time 创建时间 datetime
ip 上传时的IP string
uid 用户id int
